# Fujitsu Talent Management Portal

This app is a secure web application built with Python and Flask for managing employee talent, project allocations, and development actions within Fujitsu's internal automation team.

## Project Purpose

The aim of this project is to replace manual tracking through Microsoft tools (SharePoint, Power BI) with a structured database and consistent web application. It provides structured CRUD functionality, role-based access control, dashboard visualisations for simple metrics, and a full audit trail.
